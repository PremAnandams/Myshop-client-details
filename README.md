# Myshop-client-details

I have done the following php project
1.create a  page with client details such as Name,Email,phone,Address
2.Add a New client button to create New record in database
3.create a page to edit client details record and create new client record
4.Add a update/edit  button to update client details record
5.Add a delete record button to delete client details record.
